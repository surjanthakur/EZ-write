# InkForge AI

InkForge AI is an intelligent, modern web application designed for creators, marketers, founders, and thinkers who want to quickly transform rough ideas into high-quality written content.

## What is InkForge AI?

InkForge AI is a robust content writing platform that leverages AI to help users draft, edit, and publish compelling blogs, articles, and more—eliminating writer’s block and streamlining the writing process.

## Features

- **Smart Text Editor:** A rich, intuitive, and distraction-free editor powered by [TipTap](https://tiptap.dev/), supporting formatting options like bold, italic, underline, and strikethrough.
- **Post Type Selection:** Choose between blog or article formats for customized content workflows.
- **Seamless Content Creation:** Start writing instantly with helpful placeholders and UI feedback (toasts) for user actions.
- **Simple Publishing:** Save and publish your content with a single click, with responsive feedback and easy navigation to your dashboard.
- **Authentication & Dashboard:** (Planned/Implemented) Secure login/signup flows and a personal dashboard for managing your work.

## Technology Stack

- **Frontend:** React (Vite), TipTap Editor, Lucide Icons, React Hot Toast
- **State Management:** React Hooks (`useState`, custom hooks)
- **Routing:** React Router
- **Styling:** Tailwind CSS (utility classes), custom CSS

## How InkForge AI Works

1. **Start Writing:** The homepage allows users to jump straight into writing. The editor encourages creativity and removes friction between ideation and execution.
2. **Rich Editing Tools:** The editor comes with an accessible formatting toolbar and bubble menus for inline formatting, making content structuring effortless.
3. **Save & Publish:** Users can set a title, choose the post type, and save their content. AI-assisted tools (planned/available) help polish and elevate written drafts.
4. **Manage Content:** (Planned/Available) Access all your published drafts, blogs, and articles from a personalized dashboard.

## Who is it for?

- Writers searching for a boost in productivity
- Founders wanting to solidify ideas quickly
- Marketers creating campaign or blog content
- Anyone looking to break through creative barriers and publish with ease
